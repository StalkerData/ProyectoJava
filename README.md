# 🧭 Ruta Profesional Backend con Java

---

## 🧐 Descripción
Este repositorio documenta y contiene una ruta de aprendizaje profesional enfocada principalmente en **desarrollo backend con Java**, organizada por niveles progresivos.  
Cada nivel introduce conceptos, prácticas y proyectos diseñados para construir una base sólida antes de avanzar hacia arquitecturas más complejas, cloud y DevOps.

El enfoque principal es **Java como lenguaje backend**, priorizando lógica de negocio, buenas prácticas y escalabilidad, utilizando proyectos prácticos como evidencia del aprendizaje.

Actualmente, el repositorio se encuentra en desarrollo activo y contiene implementaciones hasta el **Nivel 1**.

---

## 🎯 Objetivo del repositorio
- Construir una base sólida en programación backend con Java.
- Aplicar conceptos teóricos mediante proyectos prácticos.
- Documentar el progreso de aprendizaje de forma estructurada.
- Servir como portafolio técnico organizado por niveles.
- Preparar el camino para tecnologías backend, cloud y DevOps en niveles posteriores.

## 🚀 Niveles de la ruta de aprendizaje

### 🏠 Nivel 1: Fundamentos Básicos
Nivel enfocado en los pilares de la programación y el backend con Java.

🏠 **[Nivel 1](Nivel_1/README.md)**

**Conceptos abordados:**
- Sintaxis básica de Java
- Variables y tipos de datos
- Operadores
- Estructuras de control (`if`, `switch`, bucles)
- Arrays y colecciones básicas
- Métodos y funciones
- Introducción a Programación Orientada a Objetos:
  - Clases
  - Objetos
- Uso de IDEs:
  - Eclipse (principal)

**Proyectos desarrollados:**
- 🧮 Calculadora en Java (consola)
- 📚 Sistema de notas en consola

Las aplicaciones de este nivel:
- Se ejecutan únicamente en consola
- No utilizan archivos ni bases de datos
- No usan frameworks
- Se enfocan en lógica y control de flujo
